## Windows 11 nastajić

* Visual Studio Code
* Docker Desktop
* Git for Windows

Potom ma so z tutym nawodom zaručić, zo je grafikowa karta w "docker" wužiwajomna: https://docs.nvidia.com/ai-enterprise/deployment/vmware/latest/docker.html
